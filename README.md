# NotebookLM Slide Deck Theme Builder (Tony Myers)

A single-file, browser-based Theme Builder for generating **NotebookLM-style slide deck prompts**.  
Includes preset themes with previews, editable design sections (Tone, Visual Identity, Image Style, Typography, Tags), and an Import tool (paste text or upload PNG/JPEG/PPTX to extract colours/fonts).

## Live App
The app is available here:

- https://tony-myers.github.io/NotebookLM-slide-deck-theme-builder/

## What this does
- **Build** a slide theme prompt by editing structured fields.
- Choose from **presets** (with thumbnails) and then tweak as needed.
- **Import**:
  - Paste a theme description to extract colours, typography, image style, and tags
  - Upload an image (PNG/JPEG) to extract dominant colours
  - Upload a PowerPoint file (.pptx) to extract theme colours and fonts

## Quick Start (recommended)
1. Open the **Live App** link.
2. Click a **preset** to load a complete style.
3. Edit any fields you want.
4. Copy the **Generated Prompt** from the right panel into your workflow.
